\#Dynamic\_AI\_voice\_agent



\#overview

It is a ai model which can be used as an calling agent. user  can upload their company data in this model in the form of a csv file and voice agent will start answering questions related to that company or will work as a calling agent for that company. i used unsupervised machine learning (clustering) to create this model.it stores data from all previous conversations and improves itself based on past conversations.
